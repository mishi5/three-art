# #112 Flash / Strobe ノード（trigger で texture を一瞬フラッシュ）

対象 Issue: https://github.com/mishi5/three-art/issues/112

親 Epic: #56 / 前提: #110（Envelope, envelopeValue）, #101 以降の texture/effect 基盤

## 目的
`onset` 等の trigger を受け、発火の瞬間に画面を一瞬フラッシュさせる effect ノード。ビート同期の閃光/ストロボ。

## 仕様
- 入力: `trigger`、任意 `in`（texture, 下地）。出力: `texture`。
- params: `release`（秒, 減衰, 既定 0.15）/ `hue`（0..1, 既定 0）/ `saturation`（0..1, 既定 0=白）。
- 挙動: trigger 立ち上がりで level=1 → `release` で 0 へ減衰（`envelopeValue(elapsed, 0, release)` 流用）。
  出力 = 下地 + flashColor·level（加算, clamp 1）。下地が無ければ黒地に flashColor·level（＝単色フラッシュ）。
- Screen 接続で表示（#98 準拠）。

## 実装
- `FlashRuntime`（純: triggerTime/prevTrigger, `feed(fired,now)` 立ち上がりエッジ検出, `getLevel(now,release)`=envelopeValue）。
- `FlashState`（GPU）: FlashRuntime + ShaderSurface(NDC_VERTEX, FRAG) + blackTexture。
- FRAG: `out = min(base + uFlash*uLevel, 1)`。`uFlash`=hsv2rgb(hue,sat,1) を CPU(THREE.Color.setHSL)で算出。
- node: category effect, isSink true（BlurNode 等に倣う）。

## TDD
- `FlashRuntime`: feed の立ち上がりエッジで triggerTime 更新（連続 true で再発火しない）、getLevel が発火直後 1・減衰・終了後 0。
- ノード定義（trigger/in 入力, texture 出力, release/hue/saturation param）。
- env 無しで no-op。
- 実描画は headless スモーク（trigger→Flash→Screen でフラッシュ、下地合成）。

## 影響
- 既存変更なし。registry の effect 群に登録。envelopeValue を EnvelopeNode から import。
